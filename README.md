# 👨‍🦳 Verificador de Idade

Este projeto é uma ferramenta interativa que calcula a idade de uma pessoa com base no ano de nascimento informado e exibe uma imagem correspondente ao sexo e à faixa etária detectada.

![Demonstração do Projeto](Captura%20de%20tela%202026-04-23%20151150.png)


## 🚀 Tecnologias Utilizadas
* **HTML5** (Estrutura)
* **CSS3** (Estilização e Layout)
* **JavaScript** (Lógica de datas e manipulação de imagens dinâmicas)

## 🔗 Demonstração
Você pode testar o projeto funcionando em tempo real clicando aqui: 
👉 [Visualizar Projeto Online](https://tamiresmires582-png.github.io/verificador-de-idade/)

## 📖 Aprendizados
Neste exercício do curso *Curso em Vídeo*, pratiquei:
- **Objeto Date**: Captura automática do ano atual do sistema.
- **Estruturas Condicionais**: Lógica para filtrar idade e sexo.
- **Manipulação de DOM**: Troca dinâmica de imagens e textos baseada na interação do utilizador.
